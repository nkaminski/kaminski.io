# Gemini CLI Context for kaminski.io

## Project Overview
This repository contains the source code for the personal portfolio and resume website of Nash Kaminski. The website showcases his background in computer engineering, professional experience, projects, and personal infrastructure.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript (e.g., Modernizr, custom scripts).
- **Assets**: Custom fonts, images.

## Engineering Standards & Mandates
- **Preserve Styling**: **DO NOT** make stylistic or cosmetic changes to the codebase unless explicitly requested by the user. Maintain the existing Vanilla CSS rules and layout structures.
- **Vanilla Web Technologies**: Stick strictly to vanilla HTML, CSS, and JavaScript. Do not introduce modern frontend frameworks, preprocessors, or libraries (e.g., React, Tailwind, Sass) unless directed.
- **Accessibility & Semantics**: Maintain and respect the existing proper semantic HTML markup (e.g., `<nav>`, `<figure>`, `<figcaption>`, and `aria-labelledby` usage).
- **No Inline Styles**: **NEVER** use inline `style="..."` attributes. Always define styles in the external CSS files within the `assets/` directory.

## Content Guidelines
- When adding or updating content, strictly follow the existing HTML structure used throughout the site.
- Ensure any added links are relative where appropriate, matching the existing navigation patterns.
